# PaidyTodo

## Prerequisites

- Node (LTS recommended)
- Npm
- Android Studio (SDK, AVD) for Android builds and emulators
- Xcode and CocoaPods for macOS
- JDK for Android builds

## Quick Start

Install dependencies:

```bash
npm install
```

Install iOS CocoaPods (macOS) and link native dependencies:

```bash
brew install cocoapods
cd <project-root>/ios
pod install
```

Run the development server and launch on a device or emulator:

```bash
npm run start
npm run android
npm run ios
```

---

## Project structure (high level)

- `src/` — application source code (screens, components, store, theme)
- `assets/` — images and static assets
- `android/`, `ios/` — native projects (managed or generated by `expo prebuild`)

---

## Architecture & state

- State is located under `src/store/`.

---

## Icons & resources

- Launcher icons include multiple raster sizes (mdpi/hdpi/xhdpi/xxhdpi/xxxhdpi) and adaptive XML layers (`foreground` / `background`).
- Android uses `mipmap/` for launcher icons to avoid scaling quality issues.
